How to change the directory (file) in Command Prompt (CMD)?

Command Name Description
CD Command change the directory or navigate to another folder from our PC.

How CD works:type cd and press enter it takes you to the top of the directory tree, like"C:drive" .
Cd \ drive name If you need to go to specific folder from the current drive, just type CD and the Subfolder must be seperated by a( \ )backslash charakter.

example: CD Desktop\Pic and press Enter.
cd .. For coming back up to tree type and Enter.
ls Shows list items in the directory.
mkdir makeing a directory in the current directory.
